# Friends Weekend

Vue 3 + Vite app with invite-only Supabase auth and a private shared media gallery.

## Quick Start
Prerequisites:
- Node `22.12.0` (see `.nvmrc`)
- npm `10.9.x`

Run locally:
```bash
nvm use
npm install
npm run dev
```

Build and preview:
```bash
npm run build
npm run preview
```

Default dev URL: `http://localhost:5173`
